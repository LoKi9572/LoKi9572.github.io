Le HTML !

L' HTML est un langage de programmation utilisé sur internet. Ce langage est utilisé pour créer des sites internet. L'acronyme signifie HyperText Markup Language, ce qui signifie en français " langage de balisage d'hypertexte ".

Le CSS !

Les CSS représentent une nouvelle façon d'appliquer des styles aux éléments HTML. Ils vous permettent de définir d'importe quelle propriété de style comme la bordure, la couleur de fond, le type de caractère, l'espace entre les lettres et bien d'autre !
